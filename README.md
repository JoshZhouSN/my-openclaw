# 工作空间文档

这是我们的协作工作空间，用于存放各类项目和文档。

## 目录结构

```
workspace/
├── projects/           # 存放开发的各个项目 (ProjectA, ProjectB, etc.)
├── documents/          # 各类文档 (安全文档、项目文档等)
└── temp/               # 临时文件
```

## 使用说明

- projects/: 存放所有开发项目，每个项目一个独立目录
- documents/: 存放各类文档资料
  - security-deployment-log.md: 安全部署操作记录
  - SECURITY.md: 安全配置指南和参考文档
- temp/: 临时文件，定期清理
